# Welteinstellungen: Branchport

## Übersicht aller Konfigurationsoptionen

| Option | Wert | Bedeutung |
|--------|------|----------|
| **BLACK_ROAD** | false | Funktion ist deaktiviert. In aktivierter Form würde diese Option spezielle Straßen-Mechaniken beeinflussen. |
| **ETERNAL_SEASON** | false | Jahreszeiten wechseln normal. Deaktiviert den Modus, der die Welt in einer einzigen Jahreszeit einfrieren würde. |
| **CONSTRUCTION_SCALING** | 100 | Normale Baugeschwindigkeit. 100 = Standard. Höhere Werte beschleunigen Konstruktionen, niedrigere verlangsamen sie. |
| **SEASON_LENGTH** | 91 | Jede Jahreszeit dauert 91 Tage. Dies entspricht einem realistischen 4-Jahreszeiten-Zyklus von ca. 364 Tagen pro Jahr. |
| **WORLD_END** | keep | Weltende-Verhalten: **keep**. Die Welt ist unbegrenzt und generiert neue Karten beim Betreten unbekannter Gebiete. |
| **ITEM_SPAWNRATE** | 1.00 | Normale Item-Spawn-Rate. 1.00 = Standard. Beeinflusst die Häufigkeit von Loot in Gebäuden und auf der Welt. |
| **SPAWN_DENSITY** | 0.50 | **50 % der normalen Monster-Dichte**. Weniger Gegner auf der Welt — angenehmer für Survival-orientiertes Gameplay. |
| **EVOLUTION_INVERSE_MULTIPLIER** | 3.00 | Monster evolve **3× langsamer** als normal. Bei 3.0 dauert Monster-Evolution dreimal länger, was frühe Ziele leichter erreichbar macht. |
| **ETERNAL_TIME_OF_DAY** | normal | Tag-Nacht-Zyklus läuft normal. Andere Optionen würden die Welt in ewiger Dunkelheit oder ewigem Tageslicht einfrieren. |
| **NPC_SPAWNTIME** | 4.00 | NPCs spawnen **4× seltener** als normal. Weniger häufige NPC-Begegnungen, höherer Fokus auf Eigenständigkeit. |
| **MONSTER_RESILIENCE** | 100 | Normale Monster-Widerstandsfähigkeit. 100 = Standard. Beeinflusst Schadensresistenzen und Haltbarkeit von Kreaturen. |
| **META_PROGRESS** | true | **Aktiviert**. Ermöglicht, dass wichtige Fortschritte (z.B. Ziele, Forschung) zwischen verschiedenen Charakteren derselben Welt erhalten bleiben. |
| **MONSTER_SPEED** | 100 | Normale Monster-Geschwindigkeit. 100 = Standard. Beeinflusst nur die numerische Basis; Individuelle Monster haben ihre eigenen Speed-Werte. |

## Charakterisierung dieser Welt

**Branchport** ist eine Welt mit folgenden Merkmalen:

- **Reduzierte Feindlichkeit**: 50% Monster-Dichte und 3× langsamere Evolution schaffen eine moderatere Herausforderung
- **Soloplay-fokussiert**: Seltene NPCs (4× weniger) fördern autonome Überlebensfähigkeiten
- **Langzeitmotivation**: 91-Tage-Jahreszeiten und Meta-Progress-Tracking ermutigen zu mehreren Charakteren und langfristigem Gameplay
- **Standard-Grundlagen**: Normale Tageszeiten, Item-Spawnrate und Baugeschwindigkeit sorgen für vorhersagbare Bedingungen

Diese Welt eignet sich gut für **Spieler, die Herausforderung mit Freiraum verbinden** möchten — weder Hardcore noch Sandbox-Mode.
