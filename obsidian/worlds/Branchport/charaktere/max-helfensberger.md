# Max Helfensberger

## Übersicht

- **Szenario:** Evacuee
- **Profession:** Mechanic
- **Konzept:** Handwerker-Überlebender — stark im frühen Crafting, robust für den ersten Run

## Traits

### Positiv

| Trait | Kosten | Begründung |
|---|---|---|
| Quick | +2 | Dauerhaft schneller — rettet Leben beim Flüchten |
| Robust Genetics | +2 | Schutz vor negativen Mutationen |
| Self-Aware | +1 | Zeigt exakte HP-Werte — wichtig für Anfänger |
| Deft | +1 | Schnelleres Waffenwechseln, kein Malus beim Nachladen |
| Fast Reader | +1 | Bücher schneller lesen = schneller skilen |

| Trait | Punkte | Konsequenz |

| Near-Sighted | +2 | Brille mitnehmen — fast kein Problem |
| Lightweight | +1 | Alkohol vermeiden — trivial |
| Poor Hearing | +1 | Kein gameplay-kritischer Nachteil |
| Squeamish | +1 | Kein Menschenfleisch essen |
